unshare

open fun unshare(ownerUserId: String!, dataId: String!, otherUserId: String!): Unit

Revoke possibility to decrypt data from other user that is not data owner. It won't be possible to decrypt such data other user's PureGrant. Note, that even if further decrypt calls will not succeed for other user, he could have made a copy of decrypted data before that call.

Parameters

ownerUserId - String!: Data owner user Id.

dataId - String!: DataId.

otherUserId - String!: User Id of user to whom access is taken away.

Exceptions

PureException - PureException
