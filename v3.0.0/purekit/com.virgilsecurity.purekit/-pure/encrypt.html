encrypt

open fun encrypt(userId: String!, dataId: String!, plainText: ByteArray!): ByteArray!

Encrypts data. This method generates keypair that is unique for given userId and dataId, encrypts plainText using this keypair and stores public key and encrypted private key. Multiple encryptions for the same userId and dataId are allowed, in this case existing keypair will be used.

Parameters

userId - String!: User Id of data owner.

dataId - String!: DataId.

plainText - ByteArray!: Plain text.

Exceptions

PureException - PureException

Return
ByteArray!: Cipher text.
